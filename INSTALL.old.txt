installer
--------

** spip *

* la traditionnelle connexion à ecrire/ puis validation de la connexion sql

* cp htaccess.txt .htaccess
 => s'inspirer du htaccess.txt qui est a la racine du SVN de seenthis


* ajouter tous les plugins définis ci-dessus

* dans configuration > avancées
	cocher GD2
	choisir URLs seenthis — message/12 people/login tag/mot-clé 

* ajouter ceci dans config/mes_options.php :

define('_ANALYTICS', '');
define('_HOST', 'st.domain.net'); # seenthis.net
define('_STATIC_HOST', _HOST); # static1.seenthis.net
define('_SHORT_HOST', 'st.domain.net/li'); # seen.li (pour les tweets)
define('_API_HTTPS', false); # ne pas forcer https sur l'API
